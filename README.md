# To-Do Vite React App

A simple To-Do List application created with Vite + React (functional components).
Features:
- Add, delete, mark complete, and edit tasks
- State managed in App component
- Components: Header, ToDoList, ToDoItem
- Uses functional components, props, and map with unique keys

## How to run locally

1. Ensure Node.js (v16+) and npm are installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```
4. Open the link shown by Vite (usually http://localhost:5173).
